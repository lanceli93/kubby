# Kubby v0.2.4 Release Notes

## Features / 新功能

- **Favorite actors** — add favorite actors support with redesigned Favorites tabs, plus favorite toggle on cast cards in movie detail page
  收藏演员功能 — 全新收藏页 Tabs 设计，电影详情页演员卡片可直接收藏
- **Continue Watching fanart cards** — use landscape fanart images for Continue Watching section
  继续观看区域使用横版剧照卡片
- **Windows data directory selection** — installer custom page lets users choose data directory during install; removes runtime change in favor of install-time selection
  Windows 安装器支持自定义数据目录，安装时选择替代运行时切换

## Fixes / 修复

- **iOS video playback** — enable HLS.js ManagedMediaSource for iOS, fixing transcoded video stopping after 6s; add `playsInline` + `disableRemotePlayback` for iOS MMS compatibility
  iOS 视频播放修复 — 启用 ManagedMediaSource，解决转码视频 6 秒后停止；添加 playsInline 属性
- **Mobile video controls** — adapt video player controls layout for mobile/iOS
  移动端视频控制栏适配
- **HLS.js type error** — remove invalid arg from `Hls.isSupported()`, fixes CI type check
  修复 Hls.isSupported() 无效参数导致的 CI 类型错误
- **Windows installer upgrade safety** — clean old program files before copying on upgrade; only delete old data files after verifying copy succeeded; improve NSIS installer robustness
  Windows 安装器升级安全性 — 升级前清理旧文件，复制验证后才删除旧数据
- **Gallery upload** — batch gallery uploads to avoid body size limit
  画廊上传分批处理，避免请求体过大
- **Gallery drag overlay** — fix offset caused by stagger animation
  修复画廊拖拽浮层因交错动画导致的偏移
- **Library card hover** — fix overlay bar sub-pixel gaps on hover
  修复媒体库卡片悬停时覆盖栏的亚像素间隙

## Style / 样式

- Recolor app icons from blue to cinema indigo
  应用图标从蓝色改为影院靛蓝色
- Reduce card border-radius from `rounded-lg` to `rounded-md`
  卡片圆角从 rounded-lg 缩小到 rounded-md
- Reduce detail page poster and glass panel radius to `rounded-xl`
  详情页海报和毛玻璃面板圆角调整为 rounded-xl
- Resolution badge: white glass background with dark text, `rounded-sm`
  分辨率标签：白色玻璃背景深色文字，rounded-sm 圆角
- Match person detail title size with movie detail (`text-3xl`)
  人物详情标题大小与电影详情统一 (text-3xl)
- Truncate movie detail title to single line
  电影详情标题截断为单行
- Increase vertical gap between poster card rows
  海报卡片行间距增大

## Docs / 文档

- Add VR/360 panoramic video player implementation plan
  添加 VR/360 全景视频播放器实现方案
- Add Windows data directory customization to architecture and skill docs
  架构文档和 skill 文档中补充 Windows 数据目录自定义说明
